将表数据导出xml

公司临时有个需求要求有个工具能够导出表的数据，输出格式为xml。
于是用了一小时写了个工具类。

一、创建 元数据表

存储 表和字段之间的关系，和字段描述

  1. CREATE TABLE [dbo].[TABLE_INFO](
  2.     [tableName] [nvarchar](50) NULL,
  3.     [columnName] [nvarchar](50) NULL,
  4.     [columnDesc] [nvarchar](50) NULL
  5. )

元数据表中的测试数据

  1. tableName   columnName  columnDesc
  2. A01 A0101   姓名
  3. A01 A0000   ID
  4. A02 A0200   职务组件
  5. A02 A0000   人员主键

二、模版文件

template.txt 

  1. <?xml version="1.0" encoding="UTF-8"?>
  2. <table>
  4.   <#list rowsList as rowInfoList>
  5.   <row type="add">
  6.     <#list rowInfoList as columnInfo>    
  7.       <${columnInfo.columnName} name="${columnInfo.columnDesc}" isattachment="false">${columnInfo.columnValue}</${columnInfo.columnName}>
  8.     </#list> 
  9.   </row>
  10.   </#list> 
  11. </table>

四、如何使用

CreateXML.java

String tableName = "TableName" 指定要导出的表数据。
右键-run

  1. public static void main(String[] args) {
  3.         // TODO 指定表面
  4.         String tableName = "TableName";
  6.         try {
  7.             // 配置模版
  8.             Configuration configuration = buildConfiguration();
  9.             Template template = configuration.getTemplate(Constant.TEMPNAME);
  11.             RowInfoUtils rowInfoUtils = new RowInfoUtils();
  12.             // 构造数据 根据表名
  13.             List<List<ColumnInfo>> rowsList = rowInfoUtils.getRowInfo(tableName);
  14.             Map<String, Object> paramMap = new HashMap<String, Object>();
  15.             paramMap.put("rowsList", rowsList);
  16.             // 生成文件
  17.             outputFile(template, paramMap, tableName + ".xml");
  18.         } catch (IOException e) {
  19.             LOG.error(e.getMessage(), e);
  20.         } catch (TemplateException e) {
  21.             LOG.error(e.getMessage(), e);
  22.         }
  23.     }
